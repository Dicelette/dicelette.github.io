Génération de modèle

Le formulaire suivant vous aidera à créer un modèle de fichier de configuration pour votre serveur. Les modalités de vérification sont les mêmes que par le bot, donc si vous avez des erreurs, vous pouvez les corriger ici plutôt que manuellement. Ensuite, il vous suffira de cliquer sur "Télécharger le modèle" pour obtenir un fichier de configuration prêt à être utilisé.

remarque

Le formulaire fourni aussi un fichier csv afin d'importer plusieurs personnages en même temps. Vous pouvez ignorer ce fichier si vous n'avez pas d'importation à faire. Pour plus d'information, vous pouvez vous référer à la documentation.
