Ce bot permet de lancer des dés et d'envoyer le résultat dans un fil de discussion. De plus, il offre la possibilité d'enregistrer un modèle de statistique pour effectuer des lancers directement, évitant ainsi de répéter les mêmes actions à chaque fois.

<FontAwesomeIcon icon="fa-solid fa-arrow-right" /> Il utilise l'API [@diceRoller](https://dice-roller.github.io/documentation/) pour effectuer les lancers de dés. Il prend également en charge la notation rollem `4#(dés)` pour les lancers "en masse" (bulk roll).

Pour les statistiques, au lieu de sauvegarder l'intégralité de la feuille, le bot enregistre le message et son identifiant. Ainsi, vous conservez un contrôle total sur vos données, puisque le bot n'a pas connaissance du contenu du message.

Le bot utilise une base de données pour stocker l'identifiant du message, l'identifiant du canal, le nom de la statistique et les noms des personnages des utilisateurs. Cette base de données est un simple fichier JSON et sera nettoyée dans les cas suivants :
- Suppression du message statistique de référence.
- Suppression du canal enregistré pour le modèle (les identifiants de référence des utilisateurs ne sont pas affectés).
- Suppression du message enregistré pour le modèle.
- Suppression du thread contenant les données utilisateurs.
- Expulsion du bot du serveur.

La base de données est utilisée uniquement si vous configurez un modèle de serveur avec `/register` ou les logs d'erreur avec `/logs`.
